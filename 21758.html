# 修改 NJK

  1. 打开主题根目录: themes\shoka\layout\_partials\
  2. 打开 layout.njk 文件
  3. 页面查找 Ctrl+F 找 loading ，动画部分是 <div id=loading>

# 修改 stylus

  1. 找到 themes\shoka\source\css\_common\components\third-party\loading.styl
  2. 修改 loading 中的 css
  3. 找不到就页面查找 Ctrl+F 找 loading

# 偏移修复

  1. 打开 themes\shoka\source\css\_common\outline\outline.styl
  2. 页面查找 Ctrl+F 找 .cat
