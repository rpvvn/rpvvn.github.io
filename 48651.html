# 基础语法

# 注释

注意：文档注释是可以提取到一个程序说明文档中
注释不会影响程序的执行，因为程序是由编译后的 class 文件来输出

# 常用数据

  1. 整数

    • 有：66，88，-88，-66

  2. 小数

    • 有：1.1，-1.2

  3. 字符

    • 有：A，O，我
    • 程序中写法：‘A’，‘O’，‘我’，需要用到单引号，且只有一个字符

  4. 字符串

    • 有：我爱你，“我爱你”，“HelloWorld”
    • 程序中必须使用双引号，内容随便写

  5. 布尔值

    • 有：真，假
    • 程序中写：false，true
    • 程序中只有这两个值，false 代表假，true 代表真

  6. 空值

    • 有：NULL
    • 程序中写作：Null
    • 在程序中是一个特殊的值，空值代表空。

  7. 特殊字符

    • \n 代表换行的意思
    • \t 代表按下了一个 tab，空出两个字符大概

# 变量

  1. 变量的定义与格式

    • 变量是用来存储一个数据的，本质是是内存中的一块区域
    • 数据类型 变量名 = 数据;

  2. 变量的优点

    • 可以用来记住要处理的数据，编写的代码更加灵活，管理代码更加方便

  3. 变量的特点

    • 变量里装的数据是可以替换的

  4. 变量的使用

    • 变量需要声明才可以使用，例：int age = 10;
    • 什么类型的变量只能存储什么类型的数据。
      • int 与 double 的区别
    • 变量存在访问范围，变量定义从第一个 {} 范围内有效。且同一个范围内定义多个变量名称不能一致。
    • 定义变量可以不赋值，但是使用时需要有一个值

由此可见变量中的数据是可以被替换的

注意：在 java 中， = 是赋值的意思，而且赋值是从右往左看的
例如：int age = 10; 意思就是将 10 的值赋值给 age

# 关键字与标识符

# 关键字


在 java 中，有一些自己用到的词有特殊作用的，如：public、class、int、double
这些关键字不能拿来作为：类名、变量名
例如：int class = 10; 这个就不能用。

# 标识符

组成

  1. 由数字、字母
  2. 下划线_
  3. 美元符号 $

要求

  1. 不能以数字开头
  2. 不能以关键字作为名字
  3. 区分大小写

规范

  1. 满足表示规则
  2. 建议用英文
  3. 有意义
  4. 首字母小写，满足驼峰模式。like: int studyNumber = 1;
