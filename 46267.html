本章记录使用 ce 修改器加速百度云盘记录，仅供学习参考。

使用旧版百度云盘，并且禁止它更新。

根据百度云盘文件来看，好像并没有后端限速，只是在本地进行限速，但是不排除以后这种方法失效，所以能用且用吧！

下载地址：

百度云盘 7.23.0.10 提取码 RPWN

CE 7.5 版本提取码 RPWN

# 正常下载

# 开始 CE 加速

保持下载，接着打开 CE7.5

选择一个 host（因为有两个嘛 有一个无效的，用来连接后端应该）点击变速精灵开启，根据需要选择倍数，我直接拉曼五百倍，然后点应用。

可以看见没有效果，说明选错了。

换一个 host，看下载速度：

有时候倍数太大反而不好，根据自己需要选择。

经过测试是可以满速的，但是我连的热点，速度就是这么多了。

# 禁止更新

自从百度云盘更新后，这个 bug 就进行了修复。所以我们还想使用的话，就需要不更新。

安装 7.23.0.10 版本百度云盘后，打开安装根目录。

可以一目了然看见，自动更新文件： Autoupdate

把里面文件删除，或者整个文件夹删除都可以。

如果你发现文件夹自己下载回来，就自己删除后，新建一个名为：aotoupdate 的文件夹，然后更改它的权限，不可编辑，让百度云盘无法自己更新回来。

# 补充说明

当你网速十兆却无法通过这个 CE 修改器来达到的时候，重新安装 7.23.0.10 版本的百度云盘再次使用 CE 修改器即可恢复满速，目前来看，是可以这么操作的。

CE 修改器需要是 7.5 版本。7.4 版本无效，不知道为什么。

# END

本次教程到此结束，不懂可以留下评论你遇到的问题。
