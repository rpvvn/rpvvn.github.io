# 基础配置

  1. 第一步
    • 点击 新建
    • 函数类型选择 web函数
    • 名称随便写，这里我取名： openAI
    • 地域选择 香港 或 其他国家
    • 时区 默认

# 函数代码

  1. 第二步
    • 选择 本地上传ZIP
    • 点击下载 ZIP

# 日记配置

  1. 第三步
    • 开启的话或有费用产生，大概几毛？一块左右？看你使用量！！！

# 高级配置

  1. 第四步
    • 从环境配置开始
    • 内存 64MB
    • 初始化超时时间 默认
    • 执行超时时间 900秒

  1. 第五步
  • 下滑至 请求多并发
  • 启用并选择 2并发
  • 其余没说的全部 默认

# 触发器配置

  1. 第六步
    • 类型 默认
    • 别名 默认
    • 请求方法选择 ANY
    • 发布环境 默认
    • 鉴权方法 免鉴权

这一步有可能像我一样：相关服务器角色异常，点击授权一下就好了。
如果不行，那就重新建一个触发器

# 地址

  • 需要去掉后面的 /release
  • 得到地址就是你的接口地址
  • 直接访问会 404

# 使用接口方法

在任何支持配置 OpenAI 域名的软件中使用它

例：

  • 使用第三方：CatGPT
  • 点击右上角的🔑图标，在接口地址中我们搭建好的地址
  • 愉快使用 OpenAI
