# 开始克隆主题

  • 首先需要你在你的博客目录下打开我们的 Git bash

小提示：themes 就是你博客的 blog 下的文件夹，至于主题名字其实你可以随便取，不过最好是英文，方便记忆

下载完成后解压放到你的博客文件夹下的 themes 目录下就 ok 了
